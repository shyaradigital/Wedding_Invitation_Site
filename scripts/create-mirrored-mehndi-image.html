Create Mirrored Mehndi Background Image

Instructions:

  1. Click "Choose File" and select public/images/Mehndi_Background.jpg
  2. Click "Create Mirrored Image" button
  3. The mirrored image will be displayed below
  4. Right-click the mirrored image and select "Save image as..."
  5. Save it as Mehndi_Background_Mirrored.jpg in the public/images/ folder
